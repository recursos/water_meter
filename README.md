# water_meter
OCR de fotografias de contadores de água.  
## Procedimento:  
1. Fotografar o mostrador do contador de água;
2. Guardar as fotografias numa pasta \consumos;
3. Correr (Windows) o script Python wmeter.py;
4. O script cria dois ficheiros: um *.csv e um *.json;
5.  Cria também ficheiros log.txt e report.txt.
6.  Os ficheiros *.csv e *.json podem ser vistos como gráficos via Excel ou Python.

## Sobre as fotografias
Para este exemplo usei 10 fotos.  
Foram tiradas com telemóvel seguuro manualmente, em más condições de luz, e bastante nal enquadradas.  
Idealmente deve ser usada uma câmara estável, sempre na mesma posição e com iluminação decente.  
As fotos devem ser feitas a intervalos regulares.
